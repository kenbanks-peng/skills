# Skills

A collection of agent skills that can be installed using [skills](https://github.com/vercel-labs/skills) or [skills-tui](https://github.com/kenbanks-peng/skills-tui).

| Skill | Description |
|-------|-------------|
| `composio` | Snapshot of the `composio-cli` skill bundled with the [Composio CLI](https://github.com/ComposioHQ). Re-sync with `composio --install-skill claude && cp -R ~/.agents/skills/composio-cli/. ~/Software/Public/skills/composio/`. |
| `frontend-slides` | A fork of [zarazhangrui/frontend-slides](https://github.com/zarazhangrui/frontend-slides). Create animation-rich HTML presentations from scratch or by converting PowerPoint files |
| `sequence-diagram` |  Generate UML sequence diagrams in D2 format from function call flows |
| `skillseekers` | Wrapper skill to use [skill seekers](https://pypi.org/project/skill-seekers/) Skill Seekers turns documentation sites, GitHub repos, PDFs, videos, notebooks, wikis, and 10+ more source types into structured knowledge assets—ready to power AI Skills (Claude, Gemini, OpenAI), RAG pipelines (LangChain, LlamaIndex, Pinecone), and AI coding assistants (Cursor, Windsurf, Cline) in minutes, not hours. |
